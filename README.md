# 📂 Aiogram Bot Template by ArtemDevvv
* Готовая, модульная и масштабируемая архитектура для создания Telegram-ботов на aiogram 3.x. Этот шаблон разработан с учётом лучших практик разработки, чтобы ваш проект оставался чистым и поддерживаемым даже при росте функционала.

## Основные преимущества:
* Масштабируемость: Легко добавлять новые хендлеры, клавиатуры или команды, не ломая существующий код.
* Читаемость: Четкое разделение логики (обработка сообщений), данных (БД), интерфейса (клавиатуры) и текстов (лексикон).
* Безопасность: Конфигурация и токены вынесены в переменные окружения (.env).
* Готовность к продакшену: Включена поддержка Docker и Docker Compose для простого деплоя.
* Удобство работы с БД: Выделенный слой для подключения и моделей, плюс middleware для автоматической передачи сессии.
* Локализация: Папка lexicon позволяет легко поддерживать многоязычность (сейчас настроено под RU).
* FSM из коробки: Папка states готова для реализации сценариев с состояниями (машина состояний).
